Detected layout issue: Sidebar overlapping with white tabs

🚨 Sidebar Menu Overlap Not Fixed
Left navigation menu visually overlaps with content area tabs, affecting user experience and operation.
Solution: 
// 1. Adjust ProLayout contentStyle
contentStyle={{
  margin: '24px 16px',
  padding: 24,
  minHeight: 'calc(100vh - 200px)'
}}

// 2. Fix sidebar width
siderWidth={208}
collapsedWidth={48}
⚠️ 页签间距不足 修复中
白色页签(Tabs)与侧边栏之间缺少足够的margin-left间距。
解决方案： 
// 为Tabs组件添加左边距
.ant-tabs {
  margin-left: 16px !important;
}

// 或者为ProPageContainer添加间距
.ant-pro-page-container {
  padding-left: 16px !important;
}
ℹ️ z-index层级冲突 已修复
组件之间的z-index设置不当，导致显示层级混乱。
解决方案： 
// 明确z-index层级
.ant-pro-sider {
  z-index: 99 !important;
}

.ant-pro-layout-header {
  z-index: 100 !important;
}

.ant-tabs-nav {
  z-index: 1 !important;
}

代码修复方案

完整的BasicLayout.tsx修复代码

// 修复后的ProLayout配置
const BasicLayout: React.FC = ({ children }) => {
  return (
    <ProLayout
      // 关键修复：内容区域间距
      contentStyle={{
        margin: '24px 16px',
        padding: 24,
        background: '#fff',
        borderRadius: 8,
        minHeight: 'calc(100vh - 200px)'
      }}

      // 固定侧边栏宽度
      siderWidth={208}
      collapsedWidth={48}

      // 确保整体布局有左边距
      style={{
        marginLeft: '16px'
      }}

      // 修复页签重叠
      menu={{
        style: {
          marginTop: '8px',
          marginBottom: '8px'
        }
      }}
    >
      {children}
    </ProLayout>
  );
};
</
